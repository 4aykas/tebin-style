# TEBIN Classic theme — design

Date: 2026-06-25
Status: approved (brainstorming)

## Problem

The 2017 *TEBIN Branding Principles & Style Guide* (`.src/Tebin_Style_Guide.pdf`,
Rev. A) is the authoritative print brand book. Decompose it into the registry as
a new theme **TEBIN Classic** (`tebin-classic`) plus the brand rules it states,
then delete the source PDF (it is not needed once decomposed).

The existing `tebin` theme is a modern *web* reinterpretation (Roboto Condensed,
charcoal surfaces, logo red `#ee3124`). TEBIN Classic is a faithful reproduction
of the print book: Pantone red + grey, Roboto / Arial, corporate-print mood. The
two are intentionally distinct and `tebin` is left untouched.

## Source decomposition (what the PDF contains)

- **Logo** — "TEBIN" wordmark, `TEB` + corner mark in red, `IN` in grey; safezone
  = height of the "B"; variants on white / red / black / dark photos (white logo);
  white rectangle backing on colored backgrounds.
- **Logo rules** — never: disproportional transform, independent element resize,
  shadows, other colors.
- **Colors** — primary red `#DA291C` (Pantone 485C), grey `#898D8D` (Pantone
  423C / Black 60%); secondary palette of 8 (RGB → hex): `#B02954`, `#A43F39`,
  `#EB807A`, `#F38B4C`, `#FBD551`, `#69B7C2`, `#B3B4B6`, `#CDCDCE`.
- **Fonts** — Roboto (Regular/Medium/Bold/Black); exception: Arial in MS Office
  documents.
- **Applications** (business card, Word, PowerPoint, printed materials,
  certificate, corporate policy) — brand *usage*, not new tokens. Out of scope.

## Decisions (from brainstorming)

- Produce: theme tokens (primary + secondary palette + fonts), classic-colored
  logo assets, and the brand rules from the guide. (All three selected.)
- Leave the existing `tebin` theme untouched (do not realign its `#ee3124`).
- Manual structured decomposition via `pymupdf` (already used to read the PDF);
  no committed `docling` pipeline — a one-shot extraction does not justify a
  heavy ML dependency, and the PDF is deleted afterward.

## New theme `themes/tebin-classic/`

- `id: "tebin-classic"`, `name: "TEBIN Classic"`, `version: "1.0.0"`.
- `description`: faithful reproduction of the 2017 TEBIN print brand book.
- `industry: ["engineering", "industrial", "b2b"]`,
  `mood: ["classic", "corporate", "print"]`.
- `source: { url: "TEBIN Branding Principles & Style Guide (2017, Rev. A)", extractedBy: "manual" }`.

  `author: "TEBIN"`, `createdAt`/`updatedAt: "2026-06-25"`.

- `color`: `brand #DA291C`, `grey #898D8D`, `ink #1A1A1A`, `topbar #FFFFFF`

  `orange #F38B4C`, `yellow #FBD551`, `teal #69B7C2`, `grey-light #B3B4B6`,

  `document ["Arial","Helvetica","sans-serif"]`.

## Brand rules (append to `rules/rules.json`, category `brand`)

All `source: "tebin brand"`. Complement the three existing brand rules; no
duplicates.

1. `brand-logo-safezone` — **MUST**: keep clear space around the logo at least
   the height of the "B" in the wordmark on all sides.
2. `brand-logo-white-rectangle-on-busy` — **SHOULD**: on colored or photographic
   backgrounds where the white logo lacks contrast, place the logo inside a white
   rectangle (e.g. sponsorship contexts).
3. `brand-logo-no-distort` — **NEVER**: never apply disproportional transforms to
   the logo or rescale its elements independently.
4. `brand-logo-no-shadow` — **NEVER**: never add shadows or other effects to the
   logo.
5. `brand-logo-no-recolor` — **NEVER**: never recolor the logo outside the
   approved palette (red, grey, all-white, all-black).
6. `brand-font-roboto-arial` — **SHOULD**: set brand text in Roboto; fall back to
   Arial where Roboto is unavailable (e.g. MS Office documents).

## Pipeline, docs, cleanup

- `pnpm build` regenerates the theme `dist/*`, `registry/index.json`, and
  `rules/dist/rules.md`.
- Update root `README.md` "Available themes" table to add `tebin-classic`
  (`test/docs.test.ts` asserts the README mentions every registry theme id).
- `pnpm validate` and `pnpm check` must pass with no drift.
- Delete `.src/Tebin_Style_Guide.pdf` and the `.src/` directory after extraction.

## Success criteria

- `list_themes()` includes `tebin-classic`; `get_theme("tebin-classic")` returns
  tokens in every format.
- `get_asset("tebin-classic", "logo-full")` returns the classic-colored SVG
  (`#DA291C` + `#898D8D`); white + corner-mark assets resolve too.
- `list_rules({category:"brand"})` returns 9 rules (3 existing + 6 new).
- `pnpm test`, `pnpm validate`, `pnpm check` all green.
- `.src/` is gone.
